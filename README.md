# Simple Tasklist
A simple tasklist that allows you to add, store and remove tasks.

## Features
* Store tasks with localStorage
* Uses <a href="https://bootswatch.com/yeti/" target="_blank">Yeti bootswatch</a>
* Clear checked tasks or all tasks
